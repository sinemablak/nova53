# Element Seçme

## Element Seçme Nedir?

JavaScript ile HTML elementlerine erişmek için element seçme yöntemleri kullanılır.

Bu sayede elementlerin içeriğini değiştirebilir, stil verebilir ve event ekleyebiliriz.

---

## Kullanılan Seçme Metodları

### querySelector()

Sayfadaki ilk eşleşen elementi seçer.

Örnek:

document.querySelector("h1");

---

### querySelectorAll()

Tüm eşleşen elementleri seçer.

document.querySelectorAll("p");

---

### getElementById()

ID değeri ile element seçer.

document.getElementById("title");

---

### getElementsByClassName()

Class adına göre element seçer.

document.getElementsByClassName("box");

---

## Bu Derste Öğrenecekleriniz

- Element seçme yöntemleri
- ID ile element seçme
- Class ile element seçme
- querySelector kullanımı
- Çoklu element seçme
