# Eventler (Olaylar)

## Event Nedir?

Event, kullanıcıların web sayfası ile etkileşim kurmasını sağlayan olaylardır.

Örneğin:

- Butona tıklamak
- Klavyeden tuşa basmak
- Mouse hareketi
- Form gönderme

JavaScript sayesinde bu olayları yakalayabilir ve işlem yaptırabiliriz.

---

## Event Listener Nedir?

Event listener, belirli bir olay gerçekleştiğinde çalışacak fonksiyonu tanımlar.

Kullanımı:

element.addEventListener("event", fonksiyon);

---

## En Çok Kullanılan Event Türleri

### click
Butona veya elemente tıklanınca çalışır.

### mouseover
Mouse element üzerine gelince çalışır.

### keydown
Klavyede tuşa basılınca çalışır.

### submit
Form gönderildiğinde çalışır.
