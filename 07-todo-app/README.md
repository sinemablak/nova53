# Todo App Projesi

## Proje Amacı

Bu projede kullanıcıların görev ekleyebildiği, silebildiği ve kaydedebildiği bir uygulama geliştireceğiz.

---

## Kullanılan Konular

- DOM Manipülasyonu
- Event Kullanımı
- Form İşlemleri
- Style Değiştirme
- LocalStorage Kullanımı

---

## Proje Özellikleri

- Görev ekleme
- Görev silme
- Görevleri listeleme
- Verileri LocalStorage'da saklama
