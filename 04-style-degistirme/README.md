# Style Değiştirme

## Style Değiştirme Nedir?

JavaScript ile HTML elementlerinin görünümünü değiştirebiliriz.

Bu sayede:

- Renk değiştirebiliriz
- Boyut değiştirebiliriz
- Gizleme / gösterme yapabiliriz
- Animasyon benzeri efektler oluşturabiliriz

---

## Style Nasıl Değiştirilir?

JavaScript'te style özelliği kullanılır.

Örnek:

element.style.color = "red";

---

## Class Ekleme ve Silme

Daha düzenli stil değişimi için classList kullanılır.

### Class ekleme

element.classList.add("aktif");

### Class silme

element.classList.remove("aktif");

### Class değiştirme

element.classList.toggle("aktif");

---

## Bu Derste Öğrenecekleriniz

- Style değiştirme
- classList kullanımı
- Toggle mantığı
- Dinamik görünüm değiştirme
