# Quiz App Projesi

## Proje Amacı

Bu projede kullanıcıya sorular soran ve doğru cevapları hesaplayan bir quiz uygulaması geliştireceğiz.

---

## Kullanılan Konular

- DOM Manipülasyonu
- Event Kullanımı
- Array ve Object Kullanımı
- State Yönetimi
- Dinamik İçerik Oluşturma

---

## Proje Özellikleri

- Soru gösterme
- Şık seçme
- Skor hesaplama
- Sonuç gösterme
