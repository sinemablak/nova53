# Form İşlemleri

## Form Nedir?

Formlar, kullanıcıdan veri almak için kullanılan HTML yapılarıdır.

Örnek kullanım alanları:

- Kayıt formu
- Giriş ekranı
- Anketler
- Todo uygulamaları

---

## Form Verisi Nasıl Alınır?

JavaScript ile input değerlerine erişebiliriz.

Örnek:

input.value

---

## Submit Event

Form gönderildiğinde submit eventi çalışır.

form.addEventListener("submit", function() {})

---

## preventDefault()

Form gönderildiğinde sayfa yenilenir.

Bunu engellemek için:

event.preventDefault();

---

## Bu Derste Öğrenecekleriniz

- Form verisi alma
- Submit eventi kullanımı
- Form doğrulama mantığı
- Kullanıcı girdisini kontrol etme
