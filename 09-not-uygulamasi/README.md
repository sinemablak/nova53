# Not Uygulaması

## Proje Amacı

Kullanıcıların başlık ve içerik ekleyebildiği bir not uygulaması geliştirmek.

---

## Kullanılan Konular

- DOM Manipülasyonu
- Form İşlemleri
- LocalStorage
- Array ve Object Kullanımı
- Dinamik Kart Oluşturma

---

## Proje Özellikleri

- Not ekleme
- Not silme
- Notları listeleme
- Verileri tarayıcıda saklama
